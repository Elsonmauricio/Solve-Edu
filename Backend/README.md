# SolveEdu Backend

Backend da plataforma SolveEdu - Conectando Talentos & Desafios

## 🚀 Tecnologias

- **Node.js** + **Express** - Servidor backend
- **Supabase** - Banco de dados
- **Auth0** - Autenticação
- **Socket.io** - Comunicação em tempo real
- **Docker** - Containerização

## 📋 Pré-requisitos

- Node.js 18+
